例<1>\Large\texttt{例<1>} 例<1>

如图，AB=BC=CD,  ∠C=80°,  ∠B=160°,AB = BC = CD,\;\angle C=80\degree ,\;\angle B=160\degree ,AB=BC=CD,∠C=80°,∠B=160°,求∠A.\angle A.∠A.

此题在我的这篇文章中已经介绍过了，但我还是把过程再来一遍。

解：

以CDCDCD为边向上作正三角形CEDCEDCED，连接AEAEAE，则 AB=BC=CD=CE=EDAB = BC = CD = CE = EDAB=BC=CD=CE=ED

∠BCE=∠BCD−∠ECD=80°−60°=20°=180°−∠B,  ∴AB∥CE\angle BCE = \angle BCD - \angle ECD = 80\degree - 60\degree = 20\degree = 180\degree - \angle B ,\;\therefore AB \parallel CE∠BCE=∠BCD−∠ECD=80°−60°=20°=180°−∠B,∴AB∥CE

又∵AB=BC=CE,  ∴  \because AB = BC = CE,\;\therefore \;∵AB=BC=CE,∴四边形ABCEABCEABCE为菱形，∴AE=……=ED.\therefore AE = …… = ED .∴AE=……=ED.

∠AED=360°−∠AEC−∠CED=360°−160°−60°=140°\angle AED = 360\degree - \angle AEC - \angle CED = 360\degree - 160\degree - 60\degree = 140\degree∠AED=360°−∠AEC−∠CED=360°−160°−60°=140°

∴∠EAD=∠EDA=(180°−140°)/2=20°\therefore \angle EAD = \angle EDA = (180\degree - 140\degree)/2 = 20\degree∴∠EAD=∠EDA=(180°−140°)/2=20°

∴∠BAD=∠BAE+∠EAD=20°+20°=40°.\therefore \angle BAD = \angle BAE + \angle EAD = 20\degree + 20\degree = 40\degree .∴∠BAD=∠BAE+∠EAD=20°+20°=40°.

法<1>\large\texttt{法<1>} 法<1>

向右构造正三角形AEFAEFAEF，易得E,B,FE,B,FE,B,F共线。

设∠EAB=∠ECB=α,  ∠BDC=∠DBC=β.\angle EAB = \angle ECB = \alpha ,\;\angle BDC = \angle DBC = \beta.∠EAB=∠ECB=α,∠BDC=∠DBC=β.

在△BEC\triangle BEC△BEC 中，∠ECB+∠EBC=∠AEB\angle ECB + \angle EBC = \angle AEB∠ECB+∠EBC=∠AEB，即α+β=∠AEB=60°;\alpha + \beta = \angle AEB = 60\degree ;α+β=∠AEB=60°;

而∠EAB+∠FAB=∠EAF\angle EAB + \angle FAB = \angle EAF∠EAB+∠FAB=∠EAF，即α+∠BAF=60°,  ∴∠BAF=β.\alpha + \angle BAF = 60\degree ,\;\therefore \angle BAF = \beta .α+∠BAF=60°,∴∠BAF=β.

∠FAB=∠EDC=β,  BA=CD,  ∠F=∠DEC=60°,  ∴△FAB≅△EDC,\angle FAB = \angle EDC =\beta ,\;BA = CD ,\; \angle F = \angle DEC = 60\degree ,\; \therefore \triangle FAB \cong \triangle EDC ,∠FAB=∠EDC=β,BA=CD,∠F=∠DEC=60°,∴△FAB≅△EDC,

∴EA=AE=ED.\therefore EA = AE = ED .∴EA=AE=ED. 得证。

法<2>\large\texttt{法<2>} 法<2>

其实把正三角形换成等腰直角三角形就可以啦awa

解：向右构造等腰直角三角形AEFAEFAEF，易得E,B,FE,B,FE,B,F共线；作GC⊥BDGC\perp BDGC⊥BD于DDD.

设∠EAB=∠ECB=α,  ∠BDC=∠DBC=β.\angle EAB = \angle ECB = \alpha ,\;\angle BDC = \angle DBC = \beta.∠EAB=∠ECB=α,∠BDC=∠DBC=β.

在△BEC\triangle BEC△BEC中，∠ECB+∠EBC=∠AEB\angle ECB + \angle EBC = \angle AEB∠ECB+∠EBC=∠AEB，即α+β=∠AEB=45°;\alpha + \beta = \angle AEB = 45\degree ;α+β=∠AEB=45°;

而∠EAB+∠FAB=∠EAF\angle EAB + \angle FAB = \angle EAF∠EAB+∠FAB=∠EAF，即α+∠BAF=45°,  ∴∠BAF=β.\alpha + \angle BAF = 45\degree ,\;\therefore \angle BAF = \beta .α+∠BAF=45°,∴∠BAF=β.

∠FAB=∠GBC=β,  BA=CB,  ∠F=∠CGB=90°,  ∴△FAB≅△GBC,  ∴AF=BG.\angle FAB = \angle GBC =\beta ,\;BA = CB ,\; \angle F = \angle CGB = 90\degree ,\; \therefore \triangle FAB \cong \triangle GBC ,\;\therefore AF = BG.∠FAB=∠GBC=β,BA=CB,∠F=∠CGB=90°,∴△FAB≅△GBC,∴AF=BG.

由于三线合一，$ DG = BG = AF.$

EA=2AF,  BD=2BF,  ∴BD=2AE.EA = \sqrt{2} AF,\;BD = 2BF ,\;\therefore BD = \sqrt{2}AE.EA=2​AF,BD=2BF,∴BD=2​AE. 得证。

另一种方法同理。

这里没用上的全等就有用了！

△DGC≅△BEC,  ∴BE=DG,\triangle DGC \cong \triangle BEC ,\;\therefore BE = DG,△DGC≅△BEC,∴BE=DG,

∴AE=AF+FE=GC+2BE=22(GE+2BE)=22BD.\therefore AE = AF + FE = GC + \sqrt{2}BE = \frac{\sqrt{2}}{2}(GE + 2BE) = \frac{\sqrt{2}}{2} BD .∴AE=AF+FE=GC+2​BE=22​​(GE+2BE)=22​​BD.
